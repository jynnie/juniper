### Loading

`onClick`s that return `Promise`s, default trigger a loading state.

  <Story name="Loading">
    <Grid colNum={2}>
      <Box>

        >
          Try Me
        </Button>
      </Box>

        <Button loading>Loading</Button>
      </Box>
    </Grid>
  </Story>

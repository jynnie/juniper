# Click Out Wrapper

You can wrap elements in a `<ClickOutWrapper onClickOut={...}/>` to fire certain functions when clicking out of an element.

<Canvas>
  <Story name="Wrapper">
    <Grid colNum={3}>
      <ClickOutWrapper onClickOut={() => console.log("Out")}>
        Wrapper
      </ClickOutWrapper>
    </Grid>
  </Story>
</Canvas>

# useClickOut

You can also use the hook `usClickOut({ onClickOut })` to not add an extra wrapper element. In that case, you should apply the ref the hook returns to the element you want to fire the function when clicking out.
